For that I opted for Science Stream in 12Th and then get admission in B.Tech Computer Science at Shaheed Bhagat Singh State University, Ferozepur.

    In Bachelor's

    In Bachelor's I was Greatly confused About From Where to Start? So I decided to Start Developement.I started a Basic Web Developement Course from the Youtube and done some Assignments.
